## Project Overview

This is a **full-stack, real-time Chess game** built with **React.js** for the front-end and **Node.js + MongoDB** for the back-end. The project demonstrates advanced web development skills, including responsive UI design, server-side logic, and real-time state management.

Players can enjoy a **smooth, interactive, and visually appealing Chess experience** across desktop and mobile devices.

---

## ‚ú® Key Features & Technical Highlights

* **Responsive Chess Board:** Built with **React.js, Bootstrap, and Material UI** for a clean and intuitive interface that adapts seamlessly to different screen sizes.  
* **Server-Side Logic:** Implemented using **Node.js** and **MongoDB** to handle move validation, store game progress, and ensure stable game flow.  
* **Real-Time Gameplay:** Reduced latency and improved state synchronization by **75%**, allowing players to see moves instantly and enjoy uninterrupted gameplay.  
* **Complete Chess Rules:** Supports all standard Chess rules including **Pawn Promotion, Castling, and En Passant**.  
* **Optimized Performance:** Efficient data handling ensures smooth interactions and accurate game state updates.

## üöÄ How to Run the Project

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/Piyushkumar030/Chess-game.git
    ```
2. **Install Dependencies:**
    ```bash
    cd Chess-game
    npm install
    ```
3. **Start the Server:**
    ```bash
    npm run server
    ```
4. **Start the Frontend:**
    ```bash
    npm start
    ```
5. **Play:** Open [http://localhost:3000](http://localhost:3000) in your browser and start a two-player game.
